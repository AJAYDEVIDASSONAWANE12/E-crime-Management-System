# E-crime-Management-System
A web-based system to manage and track online crime reports, complaints, and investigation records.
• created and built an internet framework for following investigations and reporting crimes. A item PHP and MySQL were
used to build the backend, and Admin, CID, and NCO users have role-based access.
• Admin: Assigns investigation cases and manages users.
• NCO: Registers new criminal records and FIRs.
• CID: examines cases and updates the status of investigations
